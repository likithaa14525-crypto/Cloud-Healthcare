# Cloud Healthcare Record Management System

A **cloud-based healthcare record system** that allows users to manage, upload, and view medical records securely.  
This project demonstrates a professional frontend UI for healthcare data management.

## Features
- User-friendly dashboard
- Cloud-based records management
- Secure and responsive UI
- Multiple pages: Home, Dashboard, Upload, Records

## Tech Stack
- HTML
- CSS
- JavaScript
